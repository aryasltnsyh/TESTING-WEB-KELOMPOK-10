# BOUNDARY VALUE ANALYSIS - LOGIN

**USERNAME**

| Nilai Batas | Keterangan         |
| ----------- | ------------------ |
| 4 (min-1)   | Invalid (kurang 1) |
| 5 (min)     | Valid              |
| 6 (min+1)   | Valid              |
| 14 (max-1)  | Valid              |
| 15 (max)    | Valid              |
| 16 (max+1)  | Invalid (lebih 1)  |
